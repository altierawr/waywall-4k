# Waywall 4K Config

Waywall config for 4K screens (3840x2160 pixels)

This config is inteded to be used with minecraft running in FULLSCREEN.

Features:

- Thin, eye measure and wide resizes
- Eye measurement overlay
- Search crafting friendly
- Chat mode bind for typing with a different keyboard layout without rebinds
- Toggleable mirror for reading pie directory texts easier (they're tiny on 4K screens)
- Ninjabrain supported
- Easily customizable

To customize:

- Edit values at the start and end of the file (look at comments)
